План решения

1.Загрузка изображений

Сначала программа считывает все входные изображения из папки задачи.
Особенность реализации:
на Windows стандартный cv2.imread() может не работать с кириллицей в пути;
поэтому используется безопасное чтение через:
np.fromfile
cv2.imdecode


2.Преобразование изображений

Каждое изображение переводится в два представления:
HSV — для анализа яркости и насыщенности
Grayscale — для вспомогательной обработки и визуализации
Это нужно, потому что облака удобнее выделять не в RGB, а по их визуальным свойствам.


3.Построение маски облаков

Для каждого пикселя определяется, является ли он облаком.
Используем простую эвристику:
облака обычно светлые
облака обычно малонасыщенные
Поэтому строится маска:
0 — облако
1 — полезный пиксель поверхности


4.Формирование набора полезных пикселей

После построения маски для каждого изображения остаются только те пиксели, которые можно использовать для восстановления карты.
То есть алгоритм не пытается “стереть” облака, а просто игнорирует закрытые области.


5.Сравнение пикселей из разных кадров

Для каждого пикселя итоговой карты собираются значения из всех кадров, где этот пиксель не закрыт облаком.
Дальше используется правило выбора:
если доступен только один пиксель — берём его;
если пиксели похожи — используем медиану;
если все пиксели различаются — выбираем самый тёмный, потому что облака и дымка обычно делают изображение светлее.


6.Сборка итоговой карты

После выбора лучшего значения для каждого пикселя формируется итоговое изображение местности.
Если некоторый участок был закрыт облаками на всех кадрах, то он остаётся белым.


7.Построение карты покрытия

Дополнительно формируется карта покрытия:
1 — участок удалось восстановить
0 — участок не удалось восстановить
Это позволяет оценить, насколько полно собрана карта.


8.Сохранение результатов

Программа сохраняет:
исходные изображения
маски облаков
визуализации масок
итоговую карту
карту покрытия


**Ключевая идея решения

Главная идея состоит в том, что мы не удаляем облака напрямую, а восстанавливаем изображение местности из тех кадров, где нужный участок оказался открыт.
